# VS

<details>
<summary> Release调试 </summary>
<markdown> 
```
1. 打开项目属性  
2. C++ => 常规 => 调试信息格式   选 程序数据库(/Zi)  
3. C++ => 优化 => 优化          选 禁止优化(/Od)  
4. 链接器 => 调试 => 生成调试信息 选 是(/Debug)     
```
</markdown>  
</details>  


<details>
<summary> 设置C++版本 </summary>
<markdown>
```
1. 打开项目的“属性页”对话框。
2. 选择C/C++， 选择语言。
3. 在C++语言标准中，从下拉控件中选择支持的语言标准，然后选择确定或应用以保存 您的更改。
```
</markdown>
</details>
